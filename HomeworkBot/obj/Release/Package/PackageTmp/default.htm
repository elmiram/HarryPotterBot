Harry Potter Response Bot

This bot was created as a homework for Day#1 of Microsoft DevCon School "Cognitive Services".

Special features of this bot:

  • Send your message to the Harry Potter Bot! It will respond with a morphologically similar phrase.
  • The words for phases are taken from Harry Potter series.
  • This bot understands emojis! Try sending :blush: or :heart: =)
